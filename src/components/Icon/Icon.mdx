# Icon

Icons are free to use svg's from the [Ionicons](https://ionic.io/ionicons) library.

They are converted into React components using [Chakra UI's Icon component](https://chakra-ui.com/docs/components/icon#using-the-icon-component).

<Canvas of={IconStories.IconStory} sourceState={"none"} />

<Controls of={IconStories.IconStory} />

## Usage

To pick a specific Icon, import the icons name emum `AllIcons`.

<Source
  code={`
// Example using ChevronDown

import { AllIcons, Icon } from "./Icon";

const Component = () => {
  return <Icon icon={AllIcons.ChevronDown} />;
};
`} />

A list of all currently included icons with click to copy names can be found on the [icons page](?path=/story/icon-icons--icons).
